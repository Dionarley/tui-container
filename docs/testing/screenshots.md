# Screenshots

## Generate Screenshots

### Scripts

```bash
# Project overview
bash tests/test-screenshot-project.sh

# Container states
bash tests/test-screenshot-containers.sh

# Test results
bash tests/test-screenshot-tests.sh
```

## View Screenshots

```bash
ls screenshots/
cat screenshots/01-*.txt
cat screenshots/10-*.txt
cat screenshots/20-*.txt
```

## Output

Screenshots are ASCII text files stored in `screenshots/` directory.
